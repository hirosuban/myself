# how I use claude code
- research→plan→annotate→todo→impleament→test→feedback
- researchとplanはマークダウンにかかせる。claudeのplanモードは使わない。編集追記保存ができるから
- annotationではplanにコメントを追記して、改善しまくる。創造性はここまで。
- あとはtodoして、実装するだけ
- 間違ったら修正せずにrevertしてスコープ縮小（自分でいじらない）
