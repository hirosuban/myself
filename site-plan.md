# 個人ウェブサイト構成計画

**作成日**: 2026年2月28日  
**目的**: 総合的な個人サイト  
**デザイン**: ミニマル・シンプル  
**方針**: メンテナンスが楽で、長期的に運用しやすい構成

---

## 📐 サイト構成

### 1. ホームページ (`/`)
**目的**: 訪問者に第一印象を与え、サイト全体へナビゲート

**含めるコンテンツ**:
- ヒーローセクション
  - 名前
  - キャッチフレーズ/肩書き
  - 簡潔な自己紹介（2-3行）
- 主要セクションへのクイックリンク
  - About
  - Blog
  - Interests
  - Contact
- SNSリンク（GitHub, Twitter, etc.）

**デザインポイント**:
- 余白を活かした開放的なレイアウト
- 中央揃えの構成
- スクロールで次のセクションへ誘導

---

### 2. About (`/about`)
**目的**: 詳細なプロフィールとスキルの紹介

**含めるコンテンツ**:
- 自己紹介文（詳細版）
- プロフィール写真
- スキルセット
  - プログラミング言語
  - フレームワーク・ツール
  - その他の専門性
- 経験・経歴
  - タイムライン形式
  - 学歴
  - 職歴
  - プロジェクト経験

**デザインポイント**:
- 読みやすい縦スクロール構成
- タイムラインは視覚的に
- スキルはアイコン or プログレスバーで表現

---

### 3. Blog (`/blog`)
**目的**: 記事や思考の発信

**含めるコンテンツ**:
- ブログ記事一覧（カード形式）
  - サムネイル画像
  - タイトル
  - 日付
  - 抜粋
  - タグ
- フィルタリング機能
  - カテゴリー別
  - タグ別
- 検索機能（オプション）

**記事詳細ページ (`/blog/[slug]`)**:
- 記事本文（Markdown形式）
- 公開日・更新日
- タグ
- 目次（長い記事の場合）
- 前後の記事へのナビゲーション
- シェアボタン（オプション）

**デザインポイント**:
- 読みやすいタイポグラフィ
- コードハイライト対応
- シンプルなカードデザイン

---

### 4. Interests (`/interests`)
**目的**: 趣味や興味の紹介

**含めるコンテンツ**:
- 好きなもの紹介
  - コーヒー豆研究
  - 日記へのリンク
  - その他の趣味
- 各項目の説明と外部リンク
- 写真やビジュアル要素

**デザインポイント**:
- グリッドレイアウト
- カード形式で各興味を表示
- ホバーエフェクト

---

### 5. Contact (`/contact`)
**目的**: 連絡手段の提供

**含めるコンテンツ**:
- お問い合わせフォーム
  - 名前
  - メールアドレス
  - 件名
  - メッセージ
  - 送信ボタン
- SNSリンク集
- メールアドレス（直接表示 or リンク）

**デザインポイント**:
- シンプルなフォームデザイン
- バリデーション表示
- 送信完了メッセージ

---

## 🛠️ 技術スタック（推奨）

### **Astro**
メンテナンスが楽で、個人サイトに最適。

**選定理由**:
- ✅ **静的サイト生成**: 高速で軽量
- ✅ **マークダウン対応**: ブログ記事を簡単に管理
- ✅ **コンポーネント指向**: 保守性が高い
- ✅ **最小限のJavaScript**: シンプルなサイトに最適
- ✅ **拡張性**: 必要に応じてReact/Vue等を追加可能

**補助技術**:
- **Tailwind CSS**: ユーティリティファーストのスタイリング
- **MDX**: マークダウンにインタラクティブ要素を追加（オプション）
- **Vercel/Netlify**: 簡単なデプロイ

### その他の選択肢
- **Next.js**: より複雑な機能が必要な場合
- **シンプルなHTML/CSS/JS**: 最軽量、学習コストゼロ
- **Hugo**: 超高速なSSG（Go製）

## 🎨 デザインシステム

### カラーパレット（例）
```
背景: #FFFFFF (white)
テキスト: #1A1A1A (ほぼ黒)
アクセント: #4F46E5 (Indigo) or #10B981 (Emerald)
グレー: #6B7280, #9CA3AF, #E5E7EB
```

### タイポグラフィ
- **フォント**: 
  - 見出し: Inter, Poppins, Source Sans Pro
  - 本文: Inter, Roboto, Noto Sans JP
  - コード: JetBrains Mono, Fira Code
- **サイズ階層**:
  - H1: 3rem (48px)
  - H2: 2rem (32px)
  - H3: 1.5rem (24px)
  - 本文: 1rem (16px)
  - Small: 0.875rem (14px)

### スペーシング
- セクション間: 4-6rem
- 要素間: 1-2rem
- 余白: 十分に確保

### アニメーション
- ページ遷移: フェードイン
- ホバー: 軽微なスケール変化
- スクロール: スムーズ

## 📁 ディレクトリ構造（Astro想定）

```
myself/
├── public/              # 静的ファイル
│   ├── images/
│   ├── favicon.ico
│   └── fonts/
├── src/
│   ├── components/      # 再利用可能なコンポーネント
│   │   ├── Header.astro
│   │   ├── Footer.astro
│   │   ├── BlogCard.astro
│   │   └── ContactForm.astro
│   ├── layouts/         # レイアウトテンプレート
│   │   ├── BaseLayout.astro
│   │   └── BlogLayout.astro
│   ├── pages/           # ページルーティング
│   │   ├── index.astro        # /
│   │   ├── about.astro        # /about
│   │   ├── blog/
│   │   │   ├── index.astro    # /blog
│   │   │   └── [slug].astro   # /blog/記事名
│   │   ├── interests.astro    # /interests
│   │   └── contact.astro      # /contact
│   ├── content/         # マークダウンコンテンツ
│   │   └── blog/
│   │       ├── post-1.md
│   │       └── post-2.md
│   └── styles/          # グローバルスタイル
│       └── global.css
├── astro.config.mjs     # Astro設定
├── package.json
└── README.md
```
